How can dance be used to revitalize archived Irish folklore?

Project Overview

This project aims to revitalize Irish folklore through dance. Choreography workshops with primary school children will result in performances at Luan Gallery, Athlone, Ireland. Children will create shadow dances inspired by stories from the Dúchas Schools Collection.

The project will transform archived Irish folklore into living, performative art, combining movement, spoken word, and visual art installations.
